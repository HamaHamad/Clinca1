   - Click latest deployment
   - Look for errors in build log

2. **Verify files were pushed:**
   ```bash
   # Check if app/layout.tsx exists on GitHub
   # Go to: github.com/YOUR_USERNAME/YOUR_REPO/tree/main/web/app

   ```

3. **Clear Vercel cache:**
   - Deployments → ... menu → "Clear Cache and Redeploy"

4. **Check environment variables:**
   - Settings → Environment Variables
   - Verify all 3 are set correctly
